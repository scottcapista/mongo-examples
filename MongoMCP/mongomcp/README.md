# MongoMCP

Self-contained Python package for MongoDB MCP components.

## Install

From this directory:

```bash
python -m pip install -e .
```

Or from anywhere:

```bash
python -m pip install -e /MongoMCP
```

## Package Layout

This folder uses a flat package layout where `MongoMCP` is mapped to the current directory via `tool.setuptools.package-dir`.
